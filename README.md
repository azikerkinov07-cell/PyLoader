# PyLoader

[![GitHub stars](https://img.shields.io/github/stars/azikerkinov07-cell/PyLoader?style=social)](https://github.com/azikerkinov07-cell/PyLoader)
[![License](https://img.shields.io/badge/license-MIT-green)](LICENSE)

**Fast and clean Python versions downloader** with direct links and useful tools.

![Preview](assets/screenshot.jpg)

## ✨ Features

- All Python versions from **3.5** to **3.14**
- **Auto OS detection** (Windows, macOS, Linux)
- **Direct download links** (.exe, .pkg, etc.)
- One-click **copy install commands** (`pyenv`, `brew`, `winget`)
- Modern dark theme + fully responsive
- Lightweight static site (GitHub Pages)

## 🚀 How to Use

1. Open the [website](https://azikerkinov07-cell.github.io/PyLoader/)
2. Your OS will be detected automatically
3. Click **Download** for direct installer
4. Or copy the install command for pyenv / brew / winget
